Google Hacking

Na atualidade o Google é uma das principais ferramentas de pesquisa na internet. Mas, além de se obter informações comuns sobre um determinado assunto, também é possível descobrir informações "sensíveis" sobre um determinado site, usando técnicas do Google Hacking.

O Google Hacking consiste em fazer consultas avançadas no navegador, onde se tem como objetivo conseguir informações sobre um determinado alvo. Esta é uma das principais técnicas usadas na coleta de informações.

A seguir iremos descrever algumas dessas técnicas:

Coringas do Google:

O curinga do Google, é representado pelo asterisco (*), ele é usado para substituir alguma palavra da pesquisa.

O Google limita as pesquisas a 32 palavras, caso você precise de um limite maior, você pode substituir algumas palavras pelo caractere curinga, pois o Google não conta o caractere curinga como um termo da pesquisa.

Operadores booleanos:

Os operadores booleanos são:

  • AND: equivale a letra e normal.
  • NOT: exclui uma palavra da busca, mas a melhor maneira de excluir uma palavra é antecedendo a palavra com o sinal de subtração (-).
  • OR: esse operador equivale a palavra ou. Também é possível representar esse operador usando uma barra (|).

Exemplo:

intex:password | passcode intex:username | userid | user filetype:csv

O "código" acima manda o navegador:

Localizar todas as páginas que tenham a senha (password) ou o código de entrada (passcode) no texto do documento, é das páginas encontradas, mostrar apenas as que contém as palavras username, userid ou user no texto é, entre elas mostrar apenas os documentos que sejam arquivos CSV.

Operadores avançados:

Para tornar as pesquisas mais eficientes você deve usar os operadores avançados, que são:

  • intitle, allintitle
  • inurl, allinurl
  • filetype
  • allintext
  • site
  • link
  • inanchor
  • daterange
  • cache
  • info
  • related
  • phonebook
  • bphonebook
  • author
  • group
  • msgid
  • insubject
  • stocks
  • define
  • ext

A sintaxe básica de um operador avançado é a seguinte:

operador:termo_de_pesquisa

É importante ressaltar que:

  • Não pode haver espaço entre o operador, os dois pontos e o termo da pesquisa.
  • O termo da pesquisa pode ser uma palavra ou uma expressão inserida entre aspas.
  • Os operadores booleanos e caracteres especiais (como OR e +) podem ser usados em conjunto com os operadores avançados.

Agora vamos entender melhor o que os comandos descritos anteriormente fazem.

Intitle, Allintitle: esses comandos fazem uma busca dentro do título de uma página, ou seja, pesquisa o texto que está dentro das tags <title>.

O comando intitle considera apenas a primeira palavra ou expressão para fazer a busca no título da página. Já o comando allintitle considera todas as palavras ou expressões na frente do operador.

Exemplo 01:

intitle:"index.of" "backup files"

Exemplo 02:

allintitle:"index.of" "backup files"

Allintext: esse comando localiza uma determinada palavra/texto dentro da página web. Esse comando ignora o conteúdo do título, URL e links. Este comando não deve ser combinado com outros operadores avançados.

Inurl, Allinurl: esse comando pesquisa por um determinado texto em uma URL.

Exemplo 01:

inurl:admin index

Exemplo 02:

allinurl:admin index

Site: esse operador procura por páginas que estejam hospedadas em um servidor ou domínio específico.

O Google lê os nomes dos domínios da direita para a esquerda, sendo assim, procura por site:br retornará todos os domínios que terminam em .br e pesquisar por site:com.br retornará todos os domínios terminados em .com.br.

Filetype: esse operador é usado para pesquisar por arquivos de um tipo específico, ou seja, procura por arquivos com uma determinada extensão.

Exemplo:

filetype:doc pirate

Link: esse operador procura links que levem a uma determinada página, ou seja, pesquisa por páginas vinculadas a outras páginas. No lugar de um termo de pesquisa, esse operador requer um URL ou nome de servidor como argumento. Esse operador não pode ser utilizado em conjunto com outros operadores ou termos de pesquisa.

Inanchor: esse comando pesquisa um texto dentro do texto de um link. Esse operador procura a representação em texto de um link e não a URL real. Esse comando ajuda a procurar a âncora, ou o texto exibido no link.

Exemplo:

Para encontrar o seguinte link dentro de uma tag <a>:

<a href="http://www.ertech.com.br/">Minha Página</a>

Precisamos pesquisar por:

inanchor:"Minha Página"

Cache: esse operador exibe a versão da página existente no cache. Como por exemplo:

cache:ertech.com.br

Ou

cache:www.netsec.net/content/index.jsp

Numrange: esse operador é usado para pesquisar números dentro de um intervalo. Esse comando precisa de dois parâmetros, um número baixo e um número alto, separados por um travessão.

O Google ignora símbolos (como marcadores de moeda e vírgulas), tornando assim a busca de números em uma página web mais fácil.

Esse operador também possui uma forma abreviada. Sendo necessário fornecer dois números na consulta separados por dois pontos horizontais. Como por exemplo: 1234..1453

Daterange: esse operador pesquisa por páginas publicadas dentro de um determinado intervalo de datas.

Esse operador é útil para localizar as páginas indexadas pelo Google dentro de um intervalo de datas. Quando o Google rastreia uma página, a data é mudada.

Os parâmetros desse operador devem ser definidos como um intervalo, com duas datas separadas por um travessão. As datas especificadas no operador devem estar na forma de duas datas julianas. É importante notar que a data juliana é o número de dias passados desde primeiro de Janeiro de 4713 a.C.

Info: esse operador exibe um resumo informativo de um site e fornece links de outras buscas do Google que possam estar relacionadas a ele.

Related: esse operador exibe sites determinados pelo Google como sendo relacionado a outro site.

Stocks: esse operador pesquisa por informações sobre uma determinada empresa no mercado de ações.

Define: este operador retorna definições para o termo de pesquisa especificado. Os argumentos desse operador podem ser uma palavra ou uma expressão.

Ext: esse operador é usado para excluir um determinado tipo de arquivo da pesquisa.

O operador ext não retorna resultados quando usado individualmente, sendo assim, deve ser usado em conjunto com o operador site para funcionar.

Exemplo:

site:ertech.com.br -ext:html
