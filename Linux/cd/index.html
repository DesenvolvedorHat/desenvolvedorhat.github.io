cd:

O comando cd é usado para mudar de diretório.

Sintaxe:

cd [opção] [nome ou caminho do diretório]

Exemplos:

01 - Para entrar direto no diretório home do usuário logado, você pode digitar no terminal:

cd

Ou:

cd ~

02 - Para entrar no diretório pai do diretório atual você deve usar dois pontos:

cd ..

03 - Para voltar ao diretório anterior, você deve usar um hífen:

cd -

04 - Para entrar em algum diretório você só precisa informar o cominho. Como por exemplo, vamos entrar no diretório /etc:

cd /etc

Para obter mais informações sobre o comando cd, acesse o seu manual, basta digitar no terminal:

man cd
