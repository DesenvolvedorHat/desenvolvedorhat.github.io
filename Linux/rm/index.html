rm:

O comando rm é usado para remover arquivos.

Sintaxe:

rm [opções] arquivo/diretório

Opções:

-f:

Essa opção apaga o arquivo/diretório sem pedir confirmação.

-i:

Essa opção apaga o arquivo/diretório após pedir confirmação.

-r:

Essa opção apaga arquivos e subdiretórios.

-v:

Essa opção lista arquivos deletados.

Exemplo:

01 - Para apagar um arquivo basta apenas colocar o nome do arquivo após o comando rm:

rm teste.txt

Você também pode usar curingas.

Para saber mais informações sobre o comando rm digite no terminal:

man rm
