cat:

De forma geral, o comando cat é usado para concatenar arquivos.

Sintaxe:

cat [opções] arquivos

Opções:

-n :

essa opção numera todas as linhas.

−−help :

essa opção mostra as opções do comando.

−−version :essa opção mostra informações sobre o comando.

Exemplos:

01 - Exiba na tela o conteúdo dos arquivos teste1 é teste2:

cat teste1 teste2

02 - Concatene o conteúdo dos arquivos teste1 é teste2 2 no arquivo teste3:

cat teste1 teste2 > teste3

03 - Crie um arquivo chamado teste4 usando o comando cat junto com o redirecionamento de saída:

cat > teste4

04 - Use o parâmetro -n para numerar as linhas do arquivo teste.txt

cat -n teste.txt

Para obter mais informações sobre o comando cat, digite no terminal:

man cat
