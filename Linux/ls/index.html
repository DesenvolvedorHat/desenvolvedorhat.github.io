ls:

O comando ls lista o conteúdo do diretório

Sintaxe: ls [OPÇÃO] [ARQUIVO] Opções: -a

Essa opção ignora entradas que começam com . (ponto).

-b,

Imprimir escapes de estilo C para caracteres não gráficos.

-B

Essa opção não lista entradas implícitas terminadas com ~ .

-C

Listar entradas por colunas.

--color [= QUANDO]

Essa opção colori a saída.

-d, --directory

liste os próprios diretórios, não seus conteúdos.

Exemplo:

Para listar todos os arquivos (inclusive os ocultos) do diretório, deve-se digitar:

ls -al

Para obter mais informações sobre o comando ls, digite no terminal:

man ls
