rmdir:

O comando rmdir é usado para remover diretórios vazios.

Sintaxe:

rmdir [opções] diretório

Opções:

-p, --parents:

Essa opção remove uma hierarquia de diretórios.

-v, --verbose:

Essa opção exibe informações para cada diretório processado.

--help:

Essa opção informa as opções do comando.

Exemplos:

01 - Para remover um diretório dentro do diretório atual, basta apenas digitar o nome do diretório após o comando rmdir. No exemplo a seguir iremos remover o diretório teste.

rmdir teste

02 - Para remover uma árvore de diretórios use a opção -p. Note que, caso ocorra uma falha na remoção do primeiro diretório(não está vazio), o segundo diretório também não será apagado.

rmdir -p teste/exemplo

Para obter mais informações sobre o comando rmdir digite no terminal:

man rmdir
