mkdir:

O comando mkdir é usado para criar diretórios.

Sintaxe:

mkdir [opções] diretórios

Opções:

-m, −−mode :

Essa opção define as permissões do diretório.

-p, −−parents :

Essa opção cria hierarquia de diretórios.

−−help :

Essa opção lista as opções disponíveis para o comando.

Exemplos:

01 - Para criar um diretório dentro do diretório atual, é necessário apenas digitar o nome do diretório que deve ser criado após o comando mkdir. No exemplo a seguir iremos criar o diretório teste:

mkdir teste

02 - Para criar um diretório (teste) e dentro do mesmo criar um subdiretório (exemplo), digite no terminal:

mkdir -p teste/exemplo

Você pode acessar o manual do comando mkdir digitando no terminal:

man mkdir
